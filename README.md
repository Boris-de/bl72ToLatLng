# bl72ToLatLng

> Conversion from Belgian Lambert 72 projection to WGS84 latitude/longitude in python

* * *

## Usage

```python
bl72_to_latlng(235166, 148974)  # returns {'latitude': 50.64517278981963, 'longitude': 5.572941050014241}
```
